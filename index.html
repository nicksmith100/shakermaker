This option will allow you to select a base spirit and up to three additional ingredients, providing a list of cocktails which include those ingredients (you may need to add a few further ingredients too). You can then see the full recipe and instructions. If no cocktails match the selected ingredients then you will be provided with all of the cocktails you can make with your base spirit.

Look up cocktail

If you know what you want to make, then this option allows you to look up any cocktail by name, providing you with the full recipe and instructions.

This option will provide you with a random cocktail to try, again displaying the full recipe and instructions.
